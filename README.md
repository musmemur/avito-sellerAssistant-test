# 🧪 Тестовое задание для Avito Tech Frontend (Весна 2026)

## 🚀 Запуск проекта

### 1. Запуск бэкенда
```bash
cd server
npm start
```

### 1. Запуск фронтенда
```bash
npm run dev
```

## Стек технологий

- React
- TypeScript
- Redux Toolkit (RTK)
- React Router
- Vite
- Ant Design
- Tailwind CSS
- Axios

---

## Функциональность

### Страница всех объявлений

- Отображение списка объявлений продавца
- Пагинация
- Поиск по названию объявления
- Переход на страницу объявления по клику на карточку
- Отображение основной информации в карточке

**Фильтрация:**
- по категориям
- только объявления, требующие доработки

**Сортировка:**
- по цене
- по названию
- по новизне

---

### Страница объявления

- Просмотр полной информации об объявлении
- Отображение характеристик
- Плашка **"Требуются доработки"** (если применимо)

---

### Страница редактирования объявления

- Редактирование полей объявления
- Генерация улучшенного описания с помощью LLM
- Определение рыночной цены с помощью LLM
- Сохранение изменений в `localStorage` при перезагрузке страницы

---

## Использование LLM

Для реализации LLM выбран первый вариант — локальный запуск через **Ollama**.
